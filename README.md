# AI Engineering from Scratch

A comprehensive curriculum for learning AI engineering from first principles. Build, understand, and deploy AI systems end-to-end.

## Overview

This repository contains a complete learning path from mathematical foundations to production AI systems. The curriculum is organized into 20 phases covering 503 lessons across multiple domains:

- **Math Foundations** — Linear algebra, calculus, probability, optimization
- **Machine Learning** — Classical algorithms, evaluation, and deployment
- **Deep Learning** — Neural networks, transformers, and generative models
- **Computer Vision** — Image processing, CNNs, diffusion models, and vision transformers
- **NLP** — Text processing, embeddings, transformers, and language models
- **Speech & Audio** — ASR, TTS, voice cloning, and audio models
- **LLMs** — Building, training, and optimizing large language models
- **Agents** — Autonomous systems, multi-agent swarms, and production deployment
- **Infrastructure** — Serving, scaling, observability, and compliance
- **Safety & Alignment** — Ethics, red-teaming, and responsible AI

## Getting Started

1. **Review the roadmap** — Check [CURRICULAM.md](CURRICULAM.md) for the full curriculum structure
2. **Choose your starting point** — Based on your background:
   - New to programming: Start with Phase 0 (Setup & Tooling)
   - Know Python, new to ML: Start with Phase 1 (Math Foundations)
   - Know ML, new to deep learning: Start with Phase 3 (Deep Learning Core)
   - Know deep learning, want LLMs: Start with Phase 10 (LLMs from Scratch)
   - Senior engineer, want agents: Start with Phase 14 (Agent Engineering)
